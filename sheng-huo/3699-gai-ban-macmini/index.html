3699丐版MacMini食用指南

日常
生活区
发布日期:   2023-04-02
文章字数:   2.4k
阅读时长:   8 分

谢邀@若川大佬

纯windows用户，最近入手了很火的丐版Mac mini，使用了几天，感觉还不错。

写下自己心路历程，帮助最近有类似购机需求的朋友做出选择。

需求

鲁迅曾经说过，适合自己的才是最好的

普通人买电脑，最重要的就是看清自己的需求，不然就会陷入配置和价格的循环之中。

我的需求其实很简单

看视频 写文章 最好能玩原神(偶尔要帮下班了不想动的女朋友刷日常)

Question

为什么不自己装机？

  1. 也一直有自己装机，直接高预算一步到位的想法。

  2. 但是目前显卡市场实在抱歉，自己比较急急需，618也快到了，现在装机属实不理智

为什么选择mini主机？

  1. 正好上半年有好几款比较火的mini主机，并且这些主机的性价比比自己装机高得多。

  2. 装过一台正常大小主机，摆放的时候着实费了一番功夫，之后一直想装itx类型的小主机。

  3. 以后淘汰了可以直接咸鱼，或留着当电视机顶盒。即使放箱子吃灰，也不占位置。

波折

本来直接下单了Mac mini，看了各种攻略后被劝退又下单其他品牌主机，最后回归Mac mini。

Mac mini 的劝退点

  1. Mac mini需要 4K屏才能达到最好体验

  2. MacOS 需要适应时间，需要触控板才能达到最好体验

  3. 不能玩游戏，或想玩上某游戏比较麻烦，体验衰减。

  4. 更多问题待深入体验…

这些对贫困的程序员来说并不是特别大的问题，主打一个能适应。

选择 Mac mini 的理由

  1. 完美覆盖需求，MacOS保证流畅，基于需求内存外存够用，所带的接口够用，自带音频输出。

  2. 售后完善，做工完美，开箱那一刻特别惊艳，个人感觉光做工就值2k。

  3. 支持4K 60HZ双屏输出，看视频体验很好。

  4. 为进大厂做准备，提前熟悉MacOS（不是）

  5. 折腾的那半个月，入手了同 3K+ 价位很火的小主机。配置很不错，用起来没什么问题，但是机箱漏电，换一台之后，还是漏电。为了测试两台主机是不是真的有问题，担心会不会是自己的心理原因，主动被电了十余次，属实大冤种。

主机开箱

快递盒很安全，包装为硬纸壳，内有两块M形硬纸夹层，牢牢卡住主机的包装盒。

开箱体验极佳，一个主机，一个电源线，简简单单。

感觉对这种四四方方，有棱有角，简约大气，金属手感的东西欲罢不能。

比较科技风，图片就不放了，各类视频太多，主要是怕大家上班看帖被发现。(主要是懒)

MacOS使用建议

很多可以在网上搜到视频合集，秉着不重复造轮子的原则(绝对不是因为懒)。

这里 只说个人认为比较重要，或 视频没提到的 。

程序坞菜单栏

下方的白底长条叫 程序坞 ，展示各类app，提供一个快捷打开app的渠道

程序坞中大部分默认程序，我目前还没用到，除了 访达(相当于文件夹) 启动台(存放所有app)

其他统统 右键 > 选项 > 从程序坞中保留 > 取消勾选

后续再换成自己日常要用的app

上方叫 菜单栏 ，右侧展示各类常驻app的状态，左侧是当前app提供的操作。

个性化设置推荐

取消下载免费app输入密码

系统设置 > 左上角自己头像 > 媒体与购买项目 > 免费下载项目 >始终不需要

程序坞过于占位，推荐设置为默认隐藏，鼠标移动到底部，便会展示

系统设置 > 桌面与程序坞 > 自动隐藏和显示程序坞 > 打开

桌面与程序坞中，还有很多设置，可以自行尝试，与日常使用体验密切相关。下面说一下我认为需要的设置。

打开了程序坞放大效果，自动隐藏和显示菜单-仅全屏视图下，关闭了在程序坞中显示最近使用的应用程序。

关闭Mac自然滚动，Win玩家必备

鼠标滚轮滚动，Mac与Win的效果完全相反，一个向上，一个向下，Mac称之为自然滚动，更适合触控板玩家。

第一天不知道这个设置的时候，每次我都得先反应一下才能正确滚动，这里就不勉强自己了，直接关闭！

设置 > 鼠标 > 自然滚动 > 关闭

显示文件后缀，开发必备

访达 > 设置 > 高级 > 显示所有文件扩展名

台前调度，一个很有意思的新版本功能。

但是作为前纯windows玩家，不是很习惯打开一个新app，就将当前app切到后方，而不是直接堆叠的操作。

并且左侧台前调度的展示，占用了一定的显示空间。

习惯使用command + tab进行切换,可以替代台前调度的功能。

app下载

MacOS的下载软件，我碰见了有四种情况……

第一种，直接从 App Store下载的，下载完可以直接从启动台打开

第二种，.dmg结尾的文件，双击后会出现弹窗，此时在弹窗中将图标拖入文件夹即安装完成

第三种，.app结尾的文件，直接将文件拖拽入启动台即安装完成，而不是双击

第四种，忘记是那种文件了，安装完之后桌面会有类似磁盘的驱动器，驱动器直接拖入废纸篓进行卸载

快捷键

win键 === command键

alt键 === option键

【查看隐藏文件】Cmd + Shift + .

【关闭】 command + Q 彻底退出， command + W 关闭当前(不退出)

【截屏】 command + shift + 3 截全屏， command + shift + 4 可选截屏

如想截屏后能直接通过,ctrl + V进行粘贴的话,需要加上 ctrl

【可直接粘贴的截屏】 command + shift + ctrl + 3 截全屏， command + shift + ctrl + 4 可选截屏

【神器】可显示所有快捷键,包含MacOS快捷键,以及当前所在应用的快捷键,比如VSCode快捷键

CheatSheet按住command键显示当前app快捷键

全局搜索（聚焦搜索）

command + 空格 进入聚焦搜索功能，可以搜索整台电脑文件，文件内容，甚至图片内容。

太智能了。

角快捷键

隐藏下方程序坞后有一个弊端，程序坞的唤出太慢，总感觉鼠标在屏幕底部停好几秒才出现

在设置，或 按 command + 空格，输入 角快捷键，进行设置。

其效果是，当你鼠标移动到屏幕四个角时可触发什么效果。

我的选择分别是，左下 和 右下 为启动台，左上调度中心，右上桌面。

非常好用

输入法

能适应的，我们就去适应，不能适应的，也不要对自己太狠….

中英文切换，大小写切换，全半角切换，这是写md文章经常要用到的操作。

前两项还能忍，全半角切换是真不行了，经常需要中文+半角,也不想去改快捷键，记忆快捷键。

最后下载了，搜狗输入法，一步到位，彻底轻松

命令行

更改命令行颜色，默认的命令行过于普通，不够cooooooooooool，无法激活写代码的热情。

  1. 找到命令行app，command + 空格 ，搜索 terminal，打开后在程序坞右键固定

  2. 打开 命令行，左上角菜单栏 > 终端 > 设置 > 描述文件

  3. 此处可进行外观修改，我简单的选择了 Pro主题，注意，选完后一定要点击左下角的 默认 按钮

  4. 字体也可以进行修改，注意 一定要勾选 文本中的 平滑文本 会好看很多

  5. 推荐字体大小改大一点，16 或 18

前端建议

进入VSCode官网下载VSCode，进入NodeJS官网下载NodeJS，下载git，配置ssh

这里推荐一款很好用的 git源管理包，yrn，可以npm包将下载源改为taobao源

注意，命令行大部分命令都需要加一个sudo获取权限，好像没有什么办法能取消输入sudo。

如果有的话请大佬一定在评论区告诉我！！！！

然后就是vscode一个特别好用的功能推荐，

以前有个叫 async setting的插件，可以上传vscode所有设置，包括插件，用户首选项什么的。

现在vscode自己就集成了这个功能，在左侧活动栏的下方，用github账号登录即可上传下载配置。

总结

分为主机 以及 MacOS 两部分来谈

主机外观很惊艳，配置接口各方面够用，惊喜是而且自带音响，耳机推力不错。

MacOS，确实需要熟悉成本，折腾的过程还是累的，各种查资料，最后按上面配置完成后算是舒适了。

整体使用起来非常流畅，4K效果等618再开始体验。

3699，主打看视频，刷网页，写文章，运行web程序，能折腾，Mac mini绝对是个不错的选择。

最后，有钱 MBP，没钱 Mac mini
